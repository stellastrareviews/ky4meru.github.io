## Vulnerability

People generally use the same bunch of passwords **everywhere**, mixing work and personnal usage. Sometimes, a random website is compromised and credentials are leaked. Some website are specialized in colleting these leaked sensitive information and will disclose them to you for a bit of money.

## Prerequisites

* Internet connection.
* Credit card.
* Patience.

## Exploit

Simply create and account on one or more website listed in [Useful Links](/osint/leak/#useful-links), pay depending on what you're interested in, a start looking for leaked credentials based on domain name, username, email, etc.

## Useful links

* [BreachDirectory](https://breachdirectory.org/)
* [Dehashed](https://www.dehashed.com/).
* [Snusbase](https://snusbase.com/)

## Recommendations

- [ ] Don't use personnal passwords for a work context.
- [ ] Use different passwords for every application.
- [ ] Use a passwords manager to rule them all.
